# 🎯 WordShot — 言葉で撃ち抜け

**公開URL：** https://ryuseimz.github.io/gengo-score/

## What
画像を見て、言葉だけで説明する。その言語化力をAIが採点する。**1回60秒の言語化トレーニングアプリ。**

## Why
- AI時代、**プロンプト力＝言語化力**が最重要スキルになる
- でも「言語化力」を鍛える場所がない
- ゲーム感覚で、1回60秒。通学中でも休憩中でもサクッと遊べる

## ゲームフロー（現在）
1. ランダムな画像が出る（かんたん / ふつう / むずい）
2. 60秒以内に、画像を言葉で説明（最大500文字）
3. AIが3観点で採点：**的確さ / 網羅性 / 表現力**（各0-100）
4. スコア + フィードバック表示
5. 何度でも挑戦可能

## 技術スタック
- フロント：HTML/CSS/JS（シングルファイル、フレームワークなし）
- AI採点：Google Gemini 2.5 Flash API
- ホスティング：GitHub Pages
- PWA対応（ホーム画面追加でネイティブアプリっぽく動作）

## 採点方式の進化

### 現在：AI主観採点
AIが説明文を読んで印象点をつける。手軽だがスコアの客観性に限界あり。

### 次期：画像特定方式（CLIPベース）
1. プレイヤーが画像を見て言葉で説明する
2. 元の画像 + 類似画像100枚を用意
3. AIが説明文だけを頼りに、100枚の中から元画像を特定する
4. **元画像の順位 = スコア**（1位特定 = 満点）

**メリット：**
- 「伝わったかどうか」を客観的・定量的に測定できる
- AI主観ではなく、結果ベースの採点
- 類似画像はUnsplash API等で同カテゴリから自動生成可能

### 対戦時：人間が当てる方式
- プレイヤーA：画像を見て説明文を書く
- プレイヤーB：説明文だけを読んで、候補画像群から正解を選ぶ
- Bが正解 → Aの言語化力が高い
- **ゲームとしても「伝言ゲーム」的な面白さがある**
